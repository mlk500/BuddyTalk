# Pretrained Models

Download pretrained Wav2Lip models and place them in this directory.

## Download Links

- **wav2lip_gan.pth** (Recommended for better quality):
  - https://github.com/Rudrabha/Wav2Lip/releases
  - File size: ~151 MB

- **wav2lip.pth** (Faster, lower quality):
  - https://github.com/Rudrabha/Wav2Lip/releases
  - File size: ~151 MB

## Usage

After downloading, place the model file here:

```
models/
└── wav2lip_gan.pth  (or wav2lip.pth)
```

The backend will automatically detect and use the model.
